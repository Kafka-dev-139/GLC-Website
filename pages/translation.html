Guilingchao Translation Guide

How to Translate the game (PC + Android Version)
BY Kafka (text), Kirin (extra screenshots), October 2024


Lets learn how to translate this game!

Note! This guide is for PC and Android users since IOS is not supporting overlay text apps

For PC

  1. MORT by killkimno

    MORT is a program that extracts dialogs from the screen in real time using OCR and outputs a translations using DB or machine translation.

    MORT is a program that extracts dialogs from the screen in real time using OCR and outputs a translations using DB or machine translation.

    Currently, English and Japanese translation/extraction can be extracted by default, and it can also be translated by linking with a hooking program using the save to clipboard function.

    Features:

    • Realtime translate
    • OCR - TesseractOCR , Windows OCR, NHOcr, Easy OCR
    • Machine translation - Naver Papago, Google Web, Google Sheet, ezTrans, DeepL
    • Language Patch with using DB
    • Multiple OCR areas
    • Img adjust
    MORT translation demonstration

  2. Translumo - Advanced real-time screen translator for games, hardcoded subtitles in videos, static text and etc.

    Main features:

    • High text recognition precision Translumo allows to combine the usage of several OCR engines simultaneously. It uses machine learning training model for scoring each recognized result by OCR and chooses the best one.
    • Simple interface The main idea was to make tool, that does not require manual adjustments for each case and convenient for everyday use.
    • Low latency There are several implemented optimizations to reduce impact on system performance and minimize latency between the moment a text appears and actual translation.
    • Integrated modern OCR engines: Tesseract 5.2, WindowsOCR, EasyOCR
    • Available translators: Google Translate, Yandex translate, Naver Papago, DeepL
    • Available recognition languages: English, Russian, Japanese, Chinese (simplified), Korean
    • Available translation languages: English, Russian, Japanese, Chinese (simplified), Korean, French, Spanish, German, Portuguese, Italian, Vietnamese, Thai, Turkish, Arabic

For Android

Note! Bubble Screen Translate can have a very wacky translation sometimes. Translate On Screen has ads but does a decent job translating.


  1. Bubble Screen Translate

    Bubble Screen Translate is a powerful translator that supports more than 100 languages. It can be used to translate work documents such as contracts, social media, news, chats with your friends, comics and novels, text in mobile games, movie subtitles, and much more... It helps you easily overcome all language barriers in work, study, life and entertainment.

    With Bubble Screen Translate, you can translate text on almost all apps, including popular apps such as Instagram, Facebook, X, Reddit, Quora, etc. You can translate while browsing without copying text or switching back and forth with the translation app. It also supports offline translation mode to help you save data usage.


  2. Translate On Screen

    3. Translate On Screen is a smart screen translation application that helps translate text right on the screen. This app is ideal for using social networks, playing games, reading stories in foreign languages.

    It helps you quickly translate chat messages, translate posts and comments on social networking apps, translate games, comics and foreign language documents, translate shopping apps that do not support your native language.

    Translate On Screen can translate any text on any application, with extremely simple operations, you don't need to switch back and forth between translation applications. Screen Translate eliminates all language barriers with over 100 languages supported
